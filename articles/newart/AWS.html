AWS Development

This was a module for pervasive computing where I created an interface for a experimental technology of my choice which was an audio-visual AR glasses system to visualise audio to the user with voice control. this interface was created using AWS lambda to add functionality to the interface allowing for settings and other qualities to be set and changed with connections to an Alexa Chatbot to detect and process voice commands.
