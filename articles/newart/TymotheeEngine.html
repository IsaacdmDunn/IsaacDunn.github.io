Tymothee Engine

This Module was part of a Studio Simulation where a game engine, Tymothee Engine which was planned as a 2D engine simuliar to Unity and a game which was a Stardew Valley-like farming game with tower defence feautres had to be developed with a group of 9 people with my role of the engine being the development of a directory system for a content manager, the base component system and a hierarchy view although this aspect was removed mid-development as the engine was not fully functional when the game had to be made. My role during the game's development was to create the A* pathfinding and AI targeting, main menu/ gameover screens, and small gameplay features. (Github)
