Time To Rewind

Time To Rewind is a Unity game project I made for the 2020.2 Brackey's gamejam and was made within a week using the theme: Rewind.
In this game you have to escape a dungeon and reach the end of each level by using platforms and solving puzzles while only using a special ability that lets you reverse the flow of time. (Github) and (Itch.io)
